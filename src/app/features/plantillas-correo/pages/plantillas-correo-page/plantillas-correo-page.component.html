Plantillas de correo

Personaliza el HTML (y asunto) que el sistema reemplaza al enviar correos. Los reemplazos se escriben con dos llaves, p. ej. {{NombreCliente}}. Los códigos consumo-valera, valera-qr y menu-del-dia tienen un significado fijo: deben llamarse exactamente así para vincularse al flujo correspondiente.

Plantillas reservadas por el sistema

  • consumo-valera — Tras un consumo: {{NombreCliente}}, {{FechaHora}}, {{ResumenAlmuerzos}}, {{CodigoValera}}, etc.
  • valera-qr — Al crear valera: incluir <img src="cid:qr" /> y {{UrlEscaneo}}, {{FechaVencimiento}}, etc.
  • menu-del-dia — Envío del menú: {{FechaHoy}}, {{FechaHoyLarga}} y el nombre del cliente.

No renombres un código en uso: el backend lo resuelve por el nombre exacto en la base del tenant.
